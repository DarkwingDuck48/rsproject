Что нужно сделать:

Определить основные структуры (struct) на Rust:

Project: содержит метаданные (id, название, описание, дата начала/окончания). ✅

Task: представляет одну работу (id, название, продолжительность, статус, привязанные к ней ресурсы, зависимости от других задач). ✅

Dependency: описывает связь между задачами (например, "Финиш-Старт").✅

Resource: описывает исполнителя или материал (id, название, тип).✅

Подумать о том, как задачи будут ссылаться друг на друга и на ресурсы. Использовать UUID или простые целочисленные id.

Сделать эти структуры сериализуемыми и десериализуемыми с помощью крейта serde (в форматах JSON и/или MessagePack). ✅

Реализовать базовые методы для Project: создание, добавление/удаление задачи, валидация зависимостей (например, проверка на циклические зависимости).

Написать модульные тесты для основных методов, особенно для логики валидации зависимостей.
