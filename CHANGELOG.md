# Changelog

Все значимые изменения этого проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [SemVer](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Добавлено

- Шаблоны для GitHub: bug report, feature request, pull request
- Руководство для участников (`CONTRIBUTING.md`)
- Кодекс поведения (`CODE_OF_CONDUCT.md`) — Contributor Covenant 2.0
- Конфигурация шаблонов issues (`config.yml`) — запрет пустых issues

## [0.1.0] — 2025-07-04

### Добавлено

- Отображение зависимостей между задачами в интерфейсе (#16)
- Редактирование зависимостей задач
- Лицензия Apache 2.0 (`LICENSE`)
- README: описание проекта, скриншоты интерфейса, инструкции по сборке

### Изменено

- Обновлён CI/CD: убраны суффиксы `amd64` из названий артефактов сборки

## [0.0.1] — 2025-02-22

### Добавлено

- Первая предварительная версия (`v0.0.1`)
- Создание и редактирование проектов
- Управление задачами с иерархией (групповые задачи)
- Управление ресурсами
- Назначение ресурсов с проверкой доступности
- Расчёт критического пути проекта
- Диаграмма Ганта с подсветкой критического пути
- Сохранение и загрузка проекта в формате JSON
- Базовый пользовательский интерфейс (egui/eframe)
- CI: автоматическая сборка и тестирование (`ci.yml`)
- CD: автоматическая публикация релизов (`release.yml`)
